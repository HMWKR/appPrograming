## 학습 내용

와인 분류 데이터를 대시보드 형태로 보여주는 과제입니다.  
원본 데이터 조회, 클래스별 요약, 피처 중요도, 품종 예측 함수를 하나의 앱 구조로 정리했습니다.

## 파일 구성

| 파일 | 역할 |
|---|---|
| `week11_wine_dashboard_service_explained.ipynb` | 대시보드 데이터 흐름과 예측 함수를 설명한 노트북 |
| `app.py` | 대시보드에서 사용할 데이터 조회, 요약, 예측 함수 |
| `requirements.txt` | 실행에 필요한 선택 패키지 목록 |

## 실행 확인

```bash
python app.py
```
