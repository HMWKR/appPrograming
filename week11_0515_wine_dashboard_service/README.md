## 학습 내용

2026.05.15 과제 주제인 와인 품종 분류 대시보드를 설명형 노트북과 실행 가능한 보조 코드로 정리했습니다.  
와인 데이터의 주요 피처를 정리하고, 상위 6개 피처를 입력받아 class_0, class_1, class_2 중 하나로 예측하는 흐름을 구성했습니다.

## 파일 구성

| 파일 | 역할 |
|---|---|
| `week11_0515_wine_dashboard_service_explained.ipynb` | 데이터 조회, 피처 중요도, 예측 함수 흐름을 설명한 노트북 |
| `app.py` | 대시보드에서 사용할 데이터 조회, 중요도 정리, 예측 함수 |
| `requirements.txt` | 선택 실행 환경에 필요한 패키지 목록 |

## 실행 확인

```bash
python app.py
```

## 구성 기준

- 데이터 조회: 전체 샘플과 클래스별 필터링 흐름을 함수로 분리했습니다.
- 피처 중요도: 품종 예측에 사용할 상위 6개 변수를 별도 목록으로 정리했습니다.
- 품종 예측: 입력값 검증 후 각 클래스 점수를 계산하고 가장 높은 클래스를 반환합니다.
- 대시보드 데이터: 화면에서 사용할 rows, summary, feature_importance, prediction 값을 하나의 payload로 묶었습니다.
